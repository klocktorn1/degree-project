| **Category**          | **Command**                                 | **Description**                    |
| --------------------- | ------------------------------------------- | ---------------------------------- |
| **General**           | `.help`                                     | Show all available SQLite commands |
|                       | `.quit` or `.exit`                          | Exit the SQLite shell              |
|                       | `.open mydb.db`                             | Open a database                    |
|                       | `.databases`                                | Show attached databases            |
|                       | `.save backup.db`                           | Save/backup current database       |

| **Tables**            | `.tables`                                   | List all tables                    |
|                       | `.schema`                                   | Show schema for all tables         |
|                       | `.schema users`                             | Show schema for a specific table   |
|                       | `CREATE TABLE ...`                          | Create a new table                 |
|                       | `DROP TABLE users;`                         | Delete a table                     |
|                       | `ALTER TABLE users ADD COLUMN age INTEGER;` | Add column to a table              |


| **Data Querying**     | `SELECT * FROM users;`                      | View all rows                      |
|                       | `SELECT name, email FROM users;`            | Select specific columns            |
|                       | `SELECT * FROM users LIMIT 10;`             | Limit rows                         |
|                       | `SELECT * FROM users WHERE id = 1;`         | Filter rows                        |


| **Data Modification** | `INSERT INTO users (...) VALUES (...);`     | Insert data                        |
|                       | `UPDATE users SET name="John" WHERE id=1;`  | Update data                        |
|                       | `DELETE FROM users WHERE id=1;`             | Delete data                        |


| **Table Info**        | `PRAGMA table_info(users);`                 | Show columns + types               |
|                       | `PRAGMA foreign_keys;`                      | Check if FK support is on          |
|                       | `PRAGMA foreign_keys = ON;`                 | Enable foreign keys                |


| **Formatting**        | `.headers on`                               | Show column names in results       |
|                       | `.mode column`                              | Display table-like results         |
|                       | `.mode csv`                                 | Switch output to CSV               |
|                       | `.width 15 20 30`                           | Set column widths                  |


| **Import / Export**   | `.mode csv` → `.import file.csv users`      | Import CSV into table              |
|                       | `.output file.csv` → `SELECT * FROM users;` | Export query result to CSV         |
|                       | `.output stdout`                            | Return to normal output            |

.mode column


PRAGMA foreign_keys = ON; - Add that every time you open a DB connection (or configure it once in your app).




CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT NOT NULL UNIQUE,
    email TEXT NOT NULL UNIQUE,
    firstname TEXT NOT NULL,
    lastname TEXT NOT NULL,
    password_hash TEXT NOT NULL,
    created_at TEXT DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE exercises (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    description TEXT NOT NULL
);


CREATE TABLE exercise_results (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    exercise_id INTEGER NOT NULL,
    score INTEGER,
    created_at TEXT DEFAULT CURRENT_TIMESTAMP,
    completed_at TEXT,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (exercise_id) REFERENCES exercises(id)
);
